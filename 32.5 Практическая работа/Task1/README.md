# Задача 1: Информация о фильме с кинопортала

## Описание

Проект создаёт JSON-файл с подробной информацией о кинофильме. В качестве примера используется фильм *Interstellar* (2014).

## Структура проекта

## Требования

- CMake 3.10 или выше
- Компилятор с поддержкой C++20
- CLion 2025.2.4 (рекомендуется)

- ## Содержимое JSON

JSON-файл содержит следующие обязательные поля:

- **title** — название фильма
- **country** — страна производства
- **release_date** — дата выхода
- **studio** — студия
- **production_companies** — компании-производители
- **screenplay_authors** — авторы сценария
- **director** — режиссёр
- **producers** — продюсеры
- **main_cast** — главные актёры и их персонажи
- **runtime_minutes** — длительность
- **genres** — жанры

## Функции

### `createMovieJson(const std::string& filename)`

Создаёт JSON-файл с информацией о фильме.

**Параметры:**
- `filename` — путь к файлу для сохранения

**Возвращает:** `true` при успешном создании, иначе `false`

### `validateMovieJson(const std::string& filename)`

Проверяет существование и доступность JSON-файла.

**Параметры:**
- `filename` — путь к файлу для проверки

**Возвращает:** `true` если файл валиден, иначе `false`
